# 🔐 Static Web App – User Authentication Screens

This is a simple static frontend web application with the following screens:

- 🔓 Login (index.html)
- 📝 Register
- 🔑 Forgot Password
- 🛠️ Change Password
- 🧭 Dashboard (after login)
